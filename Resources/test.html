Layout

Scope of the Project

Provide a clean and clear way of creating constraints.

Project Requirements

  1. It should cover almost all of the AutoLayout functionalities
  2. It should have a test suite and a good documentation.
  3. It should provide a way of grouping constraints creation for a single object.
  4. It should provide a way to create individual constraints.
  5. It should provide convenience functions for most common layout use cases.
  6. Syntax should be intuitive and features easy to discover.
